    HTML Tags

      As told earlier, HTML is a Markup Language and makes use of various tags to format the content. These tags are enclosed within angle braces "Tag Name". Except few tags, most of the tags have their corresponding closing tags.

      For Example, <html> has its closing tag </html> and <body> tag has its closing tag </body> tag etc.

      Above example of HTML document uses the following tags-
      Sr.No Tag of Description
      1 <!DOCTYPE...>This tag defines document type and HTML version
      2 <html>This tag encloses the complete HTML document and mainly comprises of dopcument header which is represented by <head>...</head> and document body which is repsented by <xmp><body>...</body> tags
      3 <head>This tag reprents the documents headder which can keep other HTMl tags like <tittle>, <link>
      4 <tittle>This tag is used inside the Headd Tag to mention the document title.
      5 <body>This tag represents the document's body which keeps other HTML tags like <h1>,<div>,<p> etc
      6 <h1>This tag represent the heading.
      7 <p>This tag represent a paragraph.

      To learn HTML, you will need to study various tags and understand how they behave, while formatting a textual document. Learning HTML is simple as users have to learn the usage of differnt tags in order to format the text or images to make a beautiful webpage.

      Example:

      <!DOCTYPE html> <htmml> <head> <tittle>Preserve Formatting example</tittle> </head> <body> <pre> Funtion Test Function (strText){ Alert (StrText) } </pre> </body> </htmml>
